# Generate traits correlated with latent positions

Generate traits correlated with latent positions

## Usage

``` r
generate_traits(U, corr, num_traits)
```

## Arguments

- U:

  A matrix of latent positions.

- corr:

  A numeric vector (or scalar) of correlations.

- num_traits:

  The number of traits to generate.

## Value

A matrix of traits.
